# Student Management System

A web-based student management system for educational institutions, built using Node.js, Express.js, and React.js.

## Backend

The backend is built using Node.js and Express.js, with a MySQL database for storing attendance records. The API endpoints are secured using JSON Web Tokens (JWT) and role-based access control.

## Frontend

The frontend is built using React.js, with a user-friendly interface for marking attendance, viewing attendance records, and generating PDF reports.

## Getting Started

1. Clone the repository: `git clone https://github.com/charwayyyyyy/student-management-system.git`
2. Install dependencies: `npm install`
3. Start the backend server: `npm start`
4. Start the frontend server: `npm start` (in a separate terminal window)
5. Open the frontend in your web browser: `http://localhost:3000`
